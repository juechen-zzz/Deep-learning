* tab具有命令补全与文件补齐的功能，避免打错命令或者文件名
  * 接在一串命令的第一个字段后面，则为**命令补全**
  * 接在一串命令的第二个字段后面，则为**文件补齐**
* 中断目前程序： ctrl + C
* 上下翻页：shift + 上/下
* 求助：
  * 命令 + —help
  * man + 命令
